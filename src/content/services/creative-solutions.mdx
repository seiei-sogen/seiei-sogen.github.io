## 標準的なコードでは解けない課題へ

要件が固まりきっていない領域や、最先端の技術を組み合わせる必要があるプロジェクトでも、設計から実装、技術提案まで踏み込んで取り組んできました。

### AI・3D 連携などの先端領域

- 大手電機メーカーの R&D 案件で、**AI と 3D 技術を組み合わせたアプリ**のフロントエンドを担当しました。
- 特殊な 3D ライブラリ **GaussianSplats3D** を three.js / React Three Fiber と統合し、Next.js 上で動作可能な形にカスタマイズしました。
- Vercel AI SDK の `streamObject` を参考に、JavaScript の Stream API を活用したリアルタイム反映 API を独自に実装しました。

### Coding Agent を活用した開発フロー

- 2024 年から GitHub Copilot、Claude Code、Cursor、Codex、Gemini CLI などを継続的に使用しており、通算で約 2 年の実務経験があります。
- 直近では Claude Code と classmethod/tsumiki を活用し、要件定義 → 実装計画 → 実装までを一貫して進めるスタイルで開発に取り組んでいます。
- 単なるコード生成だけでなく、曖昧な点・暗黙の前提・未決事項を LLM に抽出させ、確認事項を早い段階で整理することで手戻りを抑制しています。
- 不動産系 SaaS 案件では、Claude Code のカスタムスキル (ブランチ名生成、PR ディスクリプション生成など) を整備し、開発チームの生産性向上に貢献しました。

### 新規ゼロイチでの推進

- 人材派遣業務効率化アプリの案件では、フロントエンドとバックエンドの両方を、リポジトリのゼロからの立ち上げ・技術選定 (tanstack/react-router、Hono など) を含めて単独で進めました。
- 学習系新規サービスの案件では、新規サービス立ち上げにおいて認証・決済・アナリティクスという中核機能を、要件整理から実装まで一貫して担当しました。

### 提供できる価値

「指示通り作る」だけでなく、必要なら技術選定・プロセス設計・ツール整備にまで踏み込み、課題そのものを再定義するアプローチを取ります。
