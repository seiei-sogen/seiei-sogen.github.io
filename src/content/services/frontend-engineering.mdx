## 私の得意領域

約 12 年の Web 開発キャリアの中で最も注力してきた領域がフロントエンドです。Next.js / React / TypeScript を中心に、EC サイト、業務システム、新規サービス立ち上げ、AI・3D 連携案件まで、多様なドメインで実装を担ってきました。

### 主な技術スタック

- **Next.js**: v10 〜 v16 (App Router / Server Components) までの実務経験。
- **React / TypeScript**: `@tsconfig/strictest` や `noUncheckedIndexedAccess` を活用した型安全な設計。
- **状態管理**: Redux Toolkit、TanStack Query、XState、Jotai を案件特性に応じて使い分け。
- **フォーム・バリデーション**: React Hook Form / TanStack Form と Zod / Valibot / Yup の組み合わせ。
- **特殊領域**: three.js / React Three Fiber / GaussianSplats3D を用いた 3D 表現、Vercel AI SDK 風のストリーミング UI 実装。

### 実績の一例

- 大手電機メーカー R&D 案件では、AI と 3D 技術を組み合わせたアプリのフロントエンドを実質単独で担当。GaussianSplats3D を React Three Fiber と統合し、Next.js 上で動作する形にカスタマイズしました。
- 学習系新規サービスの立ち上げでは、Next.js v16 (App Router / Server Components) と TanStack Form + Valibot で、認証・決済・アナリティクス領域の画面実装を担当しました。
- 大規模グローバル EC サイトのリプレイス案件では、旧 PHP サイトから Next.js へのリプレイス、および複数地域向けサイトの開発・保守を担当しました。

### 提供できる価値

- **要件の曖昧さに強い実装力**: 1 行の指示や口頭の要望からでも、要件を整理して画面仕様まで落とし込める推進力。
- **保守性を意識した設計**: bulletproof-react に基づく package by feature 構成、Clean Architecture を意識したレイヤード設計を経験。
- **品質ゲートの構築**: TypeScript strictest 設定、Lint、CI ゲート、テスト導入を含む初期段階からの品質担保。
