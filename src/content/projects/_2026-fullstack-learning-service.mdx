## 担当範囲

学習系の新規サービスにおいて、認証・決済・分析を含むフロントエンドおよび一部バックエンドの実装を担当しました。要件が固まっていない段階からの整理を含め、設計・実装・品質改善まで一貫して推進しています。

### 技術的な取り組み

- **認証**: better-auth でメール認証・OAuth (Google / LINE)・パスワードリセットを実装。
- **決済**: 外部決済サービス (クレジット / キャリア決済) との XML / 文字コード処理を含む IF 連携。
- **分析連携**: Treasure Data への連携処理を実装し、データ基盤と接続。
- **UI 共通化**: Storybook で共通 UI コンポーネントのバリエーション確認を実施。
