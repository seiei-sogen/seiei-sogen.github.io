## 担当範囲

人材派遣業務を効率化するアプリケーションの API バックエンドにおいて、新規リポジトリの立ち上げと API 設計・実装を担当しました。

### 技術的な取り組み

- **新規リポジトリ立ち上げ**: Hono ベースで API をゼロから設計・実装。
- **型安全化**: Zod スキーマで境界の型付けを行い、ランタイム検証と TypeScript の型を同期。
- **テスト基盤**: Bun Test + MSW で API テスト・モック開発の足場を構築。
- **データストア**: aws-sdk で Amazon DynamoDB との接続・操作を実装。
