## 担当範囲

不動産業界向け SaaS のバックエンドにおいて、Rust 製の GraphQL API の開発・リファクタリングを担当しました。

### 技術的な取り組み

- **UseCase 層実装**: Clean Architecture のレイヤード設計に沿って、ドメインロジックを UseCase 層に集約。
- **GraphQL リファクタリング**: 既存の mutation / query を、責務分離と可読性の観点で再設計。
- **型システム活用**: Rust の型システムで境界条件をコンパイル時に検出できる構造へ。
